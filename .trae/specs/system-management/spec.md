# Sun-Panel 权限表格及系统管理功能模块 - 产品需求文档

## Overview
- **Summary**: 基于 `role_permission_matrix_v2.xlsx` 文件开发一套完整的权限表格及系统管理功能模块，包含系统Logo设置、角色权限管理、部门设置、登录页背景设置、文件上传管理、备份管理与恢复、图标管理、图库管理等8大核心功能模块。
- **Purpose**: 提供完整的系统管理能力，实现细粒度权限控制、系统配置管理、数据备份恢复等企业级功能。
- **Target Users**: 系统管理员、平台运维人员、普通用户（根据权限分配）

## Goals
- 实现完整的系统Logo管理功能，支持上传、预览、尺寸调整和持久化存储
- 基于权限矩阵文件构建角色权限管理系统，支持细粒度权限控制和实时生效
- 实现多部门层级管理，支持树形结构展示和负责人设置
- 开发登录页背景自定义功能，支持上传、预设选择和显示方式设置
- 实现多文件上传管理，支持拖拽上传和进度显示
- 开发备份管理与恢复功能，支持定时备份和多种备份模式
- 实现图标管理功能，支持分类管理和批量操作
- 开发图库管理功能，整合壁纸和图标管理

## Non-Goals (Out of Scope)
- 不实现第三方OAuth认证集成
- 不实现多语言国际化（项目已有基础框架，仅做必要扩展）
- 不实现移动端专属适配（项目已有响应式设计）
- 不实现API接口文档自动生成

## Background & Context
- 项目采用 Vue 3 + TypeScript + Naive UI 前端技术栈
- 后端采用 Go + Gin + GORM + SQLite/MySQL
- 现有系统设置通过 `SystemSetting` 模型存储在数据库中
- 用户角色目前仅支持管理员(1)和普通用户(2)两级
- 文件上传已实现基础功能，支持单文件和多文件上传

## Functional Requirements

### FR-1: 系统Logo设置功能
- **FR-1.1**: 默认Logo使用 `src/assets/logo.png` 文件
- **FR-1.2**: 文件上传组件支持选择本地PNG/JPG/JPEG格式图片，大小不超过5MB
- **FR-1.3**: 实现上传前后实时预览功能，预览区域比例与实际展示区域保持一致
- **FR-1.4**: Logo保存后持久化到系统配置，支持数据库存储与CDN路径配置
- **FR-1.5**: Logo展示在登录页面文字上方正中间区域，不同屏幕尺寸下居中显示
- **FR-1.6**: 提供Logo大小调整控件，支持50px-200px范围自定义尺寸，调整时实时预览

### FR-2: 角色权限管理模块
- **FR-2.1**: 基于 `role_permission_matrix_v2.xlsx` 配置数据构建初始权限矩阵
- **FR-2.2**: 权限修改后无需系统重启立即生效，通过前端状态更新与后端缓存刷新实现
- **FR-2.3**: 开发角色选择组件，支持从现有角色列表中选择，包含搜索过滤功能
- **FR-2.4**: 新增角色功能，表单包含角色名称、描述字段，实现名称必填和唯一性验证
- **FR-2.5**: 删除角色功能，实现二次确认对话框，已分配用户的角色不允许删除
- **FR-2.6**: 细粒度权限控制，每个模块下的权限点支持单独勾选/取消，支持全选/反选
- **FR-2.7**: 模块开关控制，每个功能模块提供启用/禁用切换，禁用时自动隐藏权限点配置
- **FR-2.8**: 实现权限配置接口，支持权限矩阵数据的获取和保存，支持批量操作
- **FR-2.9**: 实现角色管理CRUD接口，包含分页查询功能

### FR-3: 部门设置功能
- **FR-3.1**: 支持创建和管理多个层级的部门结构，实现树形结构展示
- **FR-3.2**: 实现部门的查看、创建、编辑、删除完整功能，包含表单验证和操作确认
- **FR-3.3**: 开发部门负责人设置功能，支持从用户列表中选择，支持模糊搜索

### FR-4: 登录页背景设置功能
- **FR-4.1**: 支持上传用户自定义背景图片，支持自定义图片地址（含API地址）
- **FR-4.2**: 文件格式限制为PNG/JPG/JPEG，大小不超过5MB
- **FR-4.3**: 提供系统预设背景图片库供选择，至少包含5种预设背景
- **FR-4.4**: 实现背景效果实时预览和应用保存功能
- **FR-4.5**: 支持背景显示方式设置（平铺、拉伸、居中）

### FR-5: 文件上传管理功能
- **FR-5.1**: 支持同时选择并上传多个图片文件，支持拖拽上传方式
- **FR-5.2**: 实时展示每个文件的上传进度和状态（上传中、成功、失败），失败时提供重试功能
- **FR-5.3**: 开发可视化上传进度条，准确反映当前上传进度百分比，支持取消上传操作

### FR-6: 备份管理与恢复功能
- **FR-6.1**: 支持两种备份模式：单独备份数据库、全部数据备份
- **FR-6.2**: 实现定时备份任务配置，支持查看现有定时备份任务列表
- **FR-6.3**: 支持设置备份周期和保留策略
- **FR-6.4**: 开发备份的创建、恢复、删除、导出、导入完整功能集
- **FR-6.5**: 恢复操作需二次确认
- **FR-6.6**: 新增独立的系统备份功能模块，提供备份历史记录查询

### FR-7: 图标管理功能
- **FR-7.1**: 实现单个图标查询获取接口，支持按ID和名称查询
- **FR-7.2**: 开发批量获取图标功能，支持一次获取多个图标，支持按分类批量获取
- **FR-7.3**: 实现图标分类管理和搜索功能，便于快速查找所需图标
- **FR-7.4**: 支持图标收藏功能

### FR-8: 图库管理功能
- **FR-8.1**: 在系统管理菜单下新增图库菜单，包含壁纸管理和图标管理两个子菜单
- **FR-8.2**: 实现图片上传功能，支持点击上传图片或将图片拖拽到指定上传区域
- **FR-8.3**: 支持上传图片的预览、删除、分类管理功能，支持批量操作

## Non-Functional Requirements
- **NFR-1**: 所有接口响应时间不超过2秒
- **NFR-2**: 文件上传支持最大5MB限制
- **NFR-3**: 权限修改后实时生效，缓存刷新不超过1秒
- **NFR-4**: 备份文件存储路径可配置
- **NFR-5**: 前端组件支持响应式设计，适配不同屏幕尺寸

## Constraints
- **Technical**: 
  - 前端: Vue 3 + TypeScript + Naive UI + Vite
  - 后端: Go + Gin + GORM + SQLite/MySQL
  - 现有文件上传路径配置: `config.ini` 中 `base.source_path`
- **Business**: 
  - 仅管理员角色可访问系统管理模块
  - 已分配用户的角色禁止删除
- **Dependencies**: 
  - 数据库驱动已配置（MySQL/SQLite）
  - Redis缓存已配置（可选）

## Assumptions
- **Assumption 1**: 权限矩阵文件 `role_permission_matrix_v2.xlsx` 格式正确，包含模块、权限点、角色关联数据
- **Assumption 2**: 系统已配置文件存储路径，上传文件可正常写入
- **Assumption 3**: 现有用户表结构支持角色字段扩展
- **Assumption 4**: 系统已具备基础的认证和权限拦截机制

## Acceptance Criteria

### AC-1: Logo上传与预览
- **Given**: 用户进入系统Logo设置页面
- **When**: 选择本地PNG/JPG/JPEG格式图片（不超过5MB）
- **Then**: 预览区域显示图片效果，尺寸与实际展示区域比例一致
- **Verification**: `human-judgment`

### AC-2: Logo尺寸调整
- **Given**: Logo已上传并预览显示
- **When**: 拖动尺寸调整控件在50px-200px范围内调整
- **Then**: 预览区域实时显示调整后的尺寸效果
- **Verification**: `human-judgment`

### AC-3: Logo保存与展示
- **Given**: 用户调整好Logo尺寸并点击保存
- **When**: 保存成功后返回登录页面
- **Then**: 登录页面文字上方正中间显示自定义Logo，不同屏幕尺寸下居中
- **Verification**: `human-judgment`

### AC-4: 权限矩阵初始化
- **Given**: 系统首次启动或执行初始化脚本
- **When**: 读取 `role_permission_matrix_v2.xlsx` 文件
- **Then**: 权限矩阵数据正确解析并存储到数据库
- **Verification**: `programmatic`

### AC-5: 权限实时生效
- **Given**: 管理员修改了某角色的权限配置
- **When**: 保存权限配置后刷新页面
- **Then**: 权限变更立即生效，无需系统重启
- **Verification**: `human-judgment`

### AC-6: 角色新增与验证
- **Given**: 管理员进入角色管理页面
- **When**: 填写角色名称（必填）和描述并提交
- **Then**: 名称为空时提示验证错误，名称重复时提示已存在
- **Verification**: `programmatic`

### AC-7: 角色删除限制
- **Given**: 某角色已分配给至少一个用户
- **When**: 管理员尝试删除该角色
- **Then**: 系统提示该角色已分配用户，不允许删除
- **Verification**: `programmatic`

### AC-8: 细粒度权限控制
- **Given**: 管理员在权限配置页面选择某角色
- **When**: 勾选/取消某模块下的具体权限点
- **Then**: 权限点状态正确更新，支持全选/反选功能
- **Verification**: `human-judgment`

### AC-9: 模块开关控制
- **Given**: 管理员在权限配置页面
- **When**: 禁用某功能模块
- **Then**: 该模块下所有权限点配置项自动隐藏
- **Verification**: `human-judgment`

### AC-10: 部门树形展示
- **Given**: 系统存在多个层级的部门数据
- **When**: 进入部门管理页面
- **Then**: 部门以树形结构展示，层级关系清晰
- **Verification**: `human-judgment`

### AC-11: 部门负责人设置
- **Given**: 管理员编辑部门信息
- **When**: 在负责人选择框中输入关键词搜索
- **Then**: 显示匹配的用户列表，支持模糊搜索
- **Verification**: `human-judgment`

### AC-12: 登录页背景自定义
- **Given**: 用户进入登录页背景设置页面
- **When**: 上传自定义背景图片或输入图片URL
- **Then**: 预览区域实时显示背景效果
- **Verification**: `human-judgment`

### AC-13: 预设背景选择
- **Given**: 用户进入登录页背景设置页面
- **When**: 从预设背景库中选择某背景
- **Then**: 预览区域显示选中的背景效果，至少提供5种预设
- **Verification**: `human-judgment`

### AC-14: 背景显示方式设置
- **Given**: 用户已选择背景图片
- **When**: 设置背景显示方式为平铺/拉伸/居中
- **Then**: 预览区域显示对应效果，保存后登录页生效
- **Verification**: `human-judgment`

### AC-15: 多文件拖拽上传
- **Given**: 用户进入文件上传页面
- **When**: 拖拽多个图片文件到上传区域
- **Then**: 开始上传并显示每个文件的上传状态
- **Verification**: `human-judgment`

### AC-16: 上传进度显示
- **Given**: 文件正在上传中
- **When**: 观察上传进度条
- **Then**: 进度条准确反映上传百分比，支持取消上传
- **Verification**: `human-judgment`

### AC-17: 上传失败重试
- **Given**: 某文件上传失败
- **When**: 点击重试按钮
- **Then**: 该文件重新开始上传
- **Verification**: `human-judgment`

### AC-18: 备份模式区分
- **Given**: 用户进入备份管理页面
- **When**: 选择备份模式（数据库备份/全部数据备份）
- **Then**: 系统明确显示两种模式的备份内容区别
- **Verification**: `human-judgment`

### AC-19: 定时备份配置
- **Given**: 用户进入定时备份设置页面
- **When**: 设置备份周期和保留策略
- **Then**: 定时任务配置成功保存
- **Verification**: `programmatic`

### AC-20: 备份恢复二次确认
- **Given**: 用户选择某备份文件进行恢复
- **When**: 点击恢复按钮
- **Then**: 弹出二次确认对话框，确认后执行恢复
- **Verification**: `human-judgment`

### AC-21: 图标查询与分类
- **Given**: 用户进入图标管理页面
- **When**: 按名称或分类搜索图标
- **Then**: 显示匹配的图标列表，支持批量获取
- **Verification**: `human-judgment`

### AC-22: 图标收藏功能
- **Given**: 用户在图标列表中找到目标图标
- **When**: 点击收藏按钮
- **Then**: 图标标记为已收藏，可在收藏列表中查看
- **Verification**: `human-judgment`

### AC-23: 图库菜单配置
- **Given**: 系统管理菜单已加载
- **When**: 展开系统管理菜单
- **Then**: 显示图库菜单，包含壁纸管理和图标管理子菜单
- **Verification**: `human-judgment`

### AC-24: 图库图片管理
- **Given**: 用户进入图库管理页面
- **When**: 上传图片并进行分类管理
- **Then**: 支持图片预览、删除、分类，支持批量操作
- **Verification**: `human-judgment`
